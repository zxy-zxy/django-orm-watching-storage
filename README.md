# Django watching-storage app
Application contains few views which purpose is to connect to remote server
and display data.
## Usage
* Setup requirements
```bash
pip install -r requirements.txt
```
* Update .env file (example.env provided)
    * HOST
    * PORT
    * NAME(database name)
    * USER
    * PASSWORD
    * SECRET_KEY
    * DEBUG
* Run application
```bash
python main.py
```
